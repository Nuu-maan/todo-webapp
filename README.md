## 📦 Installation

BASH
# Clone repository
git clone https://github.com/Nuu-maan/todo-webapp.git

# Install dependencies
cd todo-webapp
npm install

# Configure environment
cp .env.example .env.local

# Start development server
npm run dev
BASH
